Please create a simple React (Vite) + javascript App.
Follow the following guidlines.

1. Use React Router V7 using <BrowserRouter> <Routes>
2. Make a constant file for all API Endpoints as awell as frontend routes.
3. Use single Axios Instance for all API calls.
4. Tailwind CSS v4 (strctly v4) for styling.
5. Use shadcn/ui component library for basic UI & Components.
6. React Toastify library for toasts (create a centralized utility for it.)
7. Manage error handling using try...catch and use errorToast instance to show errors or successToast instance to show success. 
8. Store enviornment variables in .env.
9. Have a src folder.
    Inside src folder, have these folders:
    - api  (for api endpoints & axios instance)
    - components (for components)
    - pages (for pages)
    - routes (for routes)
    - utils (for utilities)
    - hooks (for hooks)
    - lib (for libraries)
    - constants (for constants)

10. No need to have:
    State management library or  cintext API
    Protected Routes 
    Any validator or provider forms

11. Use React-icons for any type of icon, for that you can install it using pnpm i react-icons
